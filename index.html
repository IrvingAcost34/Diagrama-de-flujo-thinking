⚙️ Connect Supabase first

This board needs a Supabase project to store the shared diagram. Open script.js in a text editor and fill in SUPABASE_URL and SUPABASE_ANON_KEY near the top of the file.

1. Create a free project at supabase.com
2. Open the SQL editor and run the setup script from supabase-setup.sql (included alongside this file)
3. Go to Project Settings → API, copy the Project URL and anon public key
4. Paste them into the config at the top of script.js, save, and reload this page

Full instructions are in README-supabase.md
